Про проєкт

Цей сайт розроблено для ресторану "Рататуй", який спеціалізується на французькій кухні. Мета проєкту — створити сучасний, зручний та інформативний сайт, який розкаже про ресторан, його меню та історію.

Етапи створення

Етап 1: Планування

Збір вимог, аналіз конкурентів, створення макету сайту.

Етап 2: Дизайн

Розробка дизайну сайту, підбір кольорової гами, шрифтів та інших елементів.

Етап 3: Верстка

Створення HTML-сторінок, написання CSS для стилізації.

Етап 4: Програмування

Додавання функціональності за допомогою JavaScript та PHP.

Етап 5: Тестування

Перевірка сайту на різних пристроях та браузерах, виправлення помилок.

Етап 6: Запуск

Публікація сайту на хостингу, налаштування домену.
