Про проєкт

Цей сайт–візитка розроблений для ресторану "Рататуй", який спеціалізується на французькій кухні. Мета проєкту — створити сучасний, зручний та інформативний сайт–візитку, який розкаже про ресторан, його меню та історію.

Завдання дипломної роботи

Розроблений продукт міститиме адаптивний дизайн, який забезпечує зручне користування на будь-яких пристроях: комп’ютерах, планшетах та мобільних телефонах. Розміщено детальне меню, яке включає описи страв по категоріях. Це допоможе відвідувачам швидко знайти ідеальну страву, яка відповідає їхнім смаковим уподобанням. Крім того, на сайті–візитці представлена коротка інформація про заклад. Сайт–візитка створена на WordPress з використанням власного коду (PHP, JavaScript, CSS), що забезпечує зручну навігацію та інтуїтивний інтерфейс для максимального задоволення потреб гостей.

Етапи створення

Етап 1: Планування

Збір вимог, аналіз конкурентів, створення макету сайту–візитки.

Етап 2: Дизайн

Розробка дизайну сайту–візитки, підбір кольорової гами, шрифтів та інших елементів.

Етап 3: Верстка

Створення HTML-сторінок, написання CSS для стилізації сайту–візитки.

Етап 4: Програмування

Додавання функціональності за допомогою JavaScript та PHP.

Етап 5: Тестування

Перевірка сайту–візитки на різних пристроях та браузерах, виправлення помилок.

Етап 6: Запуск

Публікація сайту–візитки на хостингу, налаштування домену.
